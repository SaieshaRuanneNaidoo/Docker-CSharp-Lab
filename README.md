What is Docker?
It's a tool that lets you package up an app with everything it needs to run (like code, settings, libraries), into a neat little box known as a "container." That way, the app runs the same way no matter whose computer or server it's on, because it's not relying on whatever happens to already be installed there. 
It's like a lunchbox.Instead of hoping the cafeteria has exactly what you need, you pack your own lunch so it's ready to go anywhere at anytime.
